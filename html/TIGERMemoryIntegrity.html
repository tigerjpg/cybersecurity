This advanced system, analyzes volatile memory in Linux, Windows, and MacOs systems for any possible malware that may exist. It detects malware by using a combination of memory forensics and memory verification constantly. Enables an individual user to monitor a specific system and provides memory acquisition to determine the root of the problem.

Features

  • Supports all Linux distributions, 32- and 64-bit x86 systems providing flexibility and ease of implementation. Also supports versions of windows 7 and above, 32- and 64-bit x86 systems, and MacOsx Snow Leopard and above.
  • Provides a configurable scanning engine for automated scans of remote systems throughout an enterprise. That requires minimum to non, user skill.
  • Allows the integration of memory forensics into enterprise security information event management systems (SIEMS).
  • Includes an extensive collection of reference software (kernels and applications). It also has a tutorial for first time users and a gui that interacts with the user to implement the software.
  • Incorporates an easy-to-use user interface I to quickly assess and interpret results. It also outputs a reference page with all the locations affected in memory.
  • Delivers output in a structured data format (JSON) to facilitate analytics.

Benefits

  • Enables visibility into the state of systems software while executing in memory.
  • Verifies the integrity of a remote system's kernel and the integrity of the executable code in all processes without doing a complete memory dump.
  • Rapidly scans thousands of systems with hundreds of gigabytes of memory.
  • Delivers malware detection using an integrity verification approach to verify that all systems software running is known and unmodified to quickly identify threats.
